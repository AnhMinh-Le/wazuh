# system_epoch

## Signature

```

field: system_epoch()
```

## Outputs

| Type | Posible values |
| ---- | -------------- |
| number | Integers between `-2^63` and `2^63-1` |


## Description

Get unix epoch time in seconds from system clock. The result operation is mapped to “field”.
If the “field” already exists, then it will be replaced.


**Keywords**

- `time`
